Epsilon DT Installation Guide

Installation of Epsilon DT.

The Epsilon DT are implemented as Eclipse plug-ins. This plug-ins are available at the Eplsilon project website. The current release version coresponds to version 1.0. The downloaded file should be named plugins.zip. To install the Epsilon DT plug-ins, first uncompress thearchive into a temporary folder of your choise. For example: C:\epsilon.

Then cut the contents of this directory and past them into the eclipse plugins directory,that is: C:\eclipse\plugins.

Installation of the EOL tools is now completed. Just (re)start the Eclipse. The temporary directory, in which the contents of the plugins.zip were extracted can now be deleted.

Important Notice:If previous versions of the Epsilon plugins are installed, then they should be removed (everything that starts with the org.epsilon prefix) prior to installing the new versions and then start Eclipse with the -clean switch to avoid Eclipse getting confused with different versions. To start Eclipse with the -clean switch just create a new text file in the main directory of eclipse, write eclipse.exe -clean and save this file as clean.bat.
