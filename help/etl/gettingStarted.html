Introduction

The Epsilon Transformation Language (ETL) is a hybrid pattern-based model transformation language. An ETL module can transform one source model to one target model using transformation rules. However, since ETL reuses EOL, it is possible that more than one models are modified during the transformation process. An ETL module consists of a pre section, a number of transformation rules, a post section and a number of user-defined operations.

To describe ETL we demonstrate a complete transformation that transforms models conforming to the OO metamodel to models conforming with the DB metamodel and a complete refinement transformation that adds getter and setter operations to an OO model. The ETL files, metamodels and exemplar models for both transformations can be obtained here.
