Introduction

The Epsilon Comparison Language (ECL) is a hybrid pattern-based model comparison language. An ECL module can compare and check the conformance of two models. It consists of a pre section, a number of match rules, a post section and a number of user-defined operations. It can also use a matching strategy to further automate common comparison scenarios (e.g. comparing models that are versions of a common ancestor model)

To present ECL we demonstrate a complete comparison specification that compares a model conforming with the OO metamodel with a model conforming with the DB metamodel.
