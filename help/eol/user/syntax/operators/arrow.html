-> operator

In EOL, the -> (arrow) operator is used to invoke operations on instances (but not to access features). It is used mainly for compatibility with OCL and when called, it examines the available operations in the following order (reverse of the . operator)

1. Built-in operation
2. User defined operation

Syntax

expression->operationCall(parameters)

Examples

a->print();

Invokes the built-in print() operation on instance a.
a->doSomething();

Invokes the user-defined doSomething() operation on instance a.
