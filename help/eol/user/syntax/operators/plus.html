+ operator

The + operator is a binary operator that serves a dual purpose. When applied to items of Integer or Real type it returns their sum. When applied to anything else, it returns the concatenation of their string representations.

Syntax

operand1 + operand2

Examples

1 + 2 returns 3
1 + 1.5 returns 2.5
'Hello ' + 'World' returns 'Hello World'
'Five is ' + 5 returns 'Five is 5'
