For statement

The for statement is used to iterate collections of items. In the body of a for loop, built-in convinience variables hasMore : Boolean and loopCount : Integer are available. The first determines if the loop will be executed more times and the second how many times the body of the loop has been executed so far.

Syntax

for (iterator in collection) {
  body
}

Examples

for (i in Sequence{1..10}) {
  i.println();
}

Iterate a sequence containing integers from 1 to 10 and print each number
