Var statement

The var statement is used to define a new variable. A variable can be of either a primitive type (String,Integer,Boolean,Real), a collection type (Collection,Sequence,Set,OrderedSet) or a model element type (e.g. Uml!Class).

Syntax

var variableName : (new)* variableType;

Examples

var i : Integer;

Defines the variable i which is of type Integer
var class : new Uml!Class;

Defines the variable class which is of type Uml!Class The new keyword states that the variable is assigned to a new instance of the Uml!Class metaclass.
var seq : Sequence(Uml!Class);

Defines the variable seq which is a Sequence of elements of Uml!Class
