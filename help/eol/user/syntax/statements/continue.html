Continue statement

The continue statement is used to disgard the remainted of the body of a for or a while loop and procced to the next iteration.

Syntax

continue ;

Examples

for (i in Sequence{1..10}){
  if (i = 5){
    continue;
  }
  else {
    i.println();
  }
}

This loop will print numbers 1-4 and 6-10 but not 5
