Return statement

The return statement is used to (optionally) return a value and exit a statement block. In EOL, it is valid only in the body of a user defined operation but in languages built atop EOL it is also used in bodies of rules.

Syntax

return (value)?;

Examples

operation Integer add(i : Integer ) : Integer{
  return self + i;
}

Returns the sum of the Integer on which it was invoked (accessible via the self built-in variable) with i.
