If statement

The if statement is used to implmement logical branching. The booleanExpression is evaluated and if it returns true the body of the if is executed, otherwise the body of the else (if an else clause has been defined) is executed.

Syntax

if (booleanExpression) {
  body
}
(else {
  body
})?

Examples

if (x < 0) {
  'x is negative'.println();
}
else {
  'x is non-negative'.println();
}

Checks if the value of the variable x is less than zero and print a suitable message for each case.
