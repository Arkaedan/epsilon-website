While statement

The while statement is used facilitate non-collection-based loops. In the body of a while loop, the built-in convinience variable loopCount : Integer is available to determine how many times the body of the loop has been executed so far.

Syntax

while (condition) {
  body
}

Examples

var i : Integer;
i := 0;
while (i<3){
  i := i + 1;
  i.println();
}

Prints numbers 1-3
