Model datatype

The Model datatype represents models accessible from EOL. In EOL, references to models are aquired through their names.

Built-in operations

createInstance(type : String) : Any Creates a new instance of the type in the model
store() : Boolean Stores the model to the file from which it was loaded
store(newLocation : String) : Boolean Stores the model to the file specified (absolute path)
allInstances() : Sequence Returns all the model elements in the model
deleteInstance(instance : Any) : Boolean Deletes the specified instance from the model
owns(instance : Any) : Boolean Checks if the specified instance is owned by the model
hasType(typeName : String) : Boolean Checks if the model has a model-element type with this name
