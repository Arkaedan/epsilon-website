String datatype

The String datatype represents a sequence of characters. Litteral strings are expressed as sequences of characters enclosed in single quotes (') (e.g. 'I am a litteral string') and the escape character is the backslash (\)

Built-in operations (on instances of the type)

firstToUpperCase() : String Returns a copy of the string but with the first letter converted to its upper-case equivalent
firstToLowerCase() : String Similar to the previous but for lower-case instead
toUpperCase() : String Returns a copy of the string with each character converted to its upper-case equivalent
toLowerCase() : String Similar to the previous but for lower-case instead
startsWith(start : String) : Boolean Checks if the string starts with start
endsWith(end : String) : Boolean Checks if the string ends with end
isSubstringOf(str : String) : Boolean Checks if the string is a substring of str
toCharSequence() : Sequence Returns a sequence that contains the characters of the string
replace(what : String, with : String) : String Returns a copy of the string after replacing (using regular expression) substrings that match to the what parameter with value of the with parameter
charAt(position : Integer) : String Returns a string that contains the character of the string at position
substring(startIndex : Integer, endIndex : Integer) : String Returns the substring starting at startIndex and ending at endIndex
split(delimiter : String) : Sequence Splits the string into a sequence of sub-strings
trim() : String Returns a copy of the string without leading and trailing spaces
matches(regExp : String) : String Returns true if the string matches the regular expression regExp
