Enumeration types

Many modelling technologies provide the concept of Enumeration types. Each enumeration-type has a finite number of primitive literals. To access a literal, the # operator can be used. For example, to aquire the value of the vk_public literal of the VisibilityKind in a UML 1.4 model named Uml, the Uml!VisibilityKind#vk_public expression can be used. As with model-element types, in case of conflicting names, the fully qualified path of the enumeration-type can be used.

Syntax

modelName!enumerationType#literalName

or

modelName!package1::package2::enumerationType#literalName

Examples

Uml!VisibilityKind#vk_public

Returns the value of the vk_public literal of the VisibilityKind enumeration type in the Uml model.
