Map datatype

The Map datatype represents an object with dynamic features. The user can assign and retrieve arbitrary feature values.

Examples

var m : Map;
m.someFeature := 'testValue';
m.someOtherFeature := 1;
(m.someFeature + m.someOtherFeature).println();

In the above example, variable m is defined to be a Map. The definition automatically creates a new instance of Map and assigns it to the variable. At that time, m has no features. Upon executing the first assignment statement, m creates a feature named someFeature and assigns the 'testValue' value to it. Similarly, the second assignment creates a someOtherFeature feature and sets its value to the Integer 1. Both features are accessed in the final statement which prints testValue1 to the default output stream.
