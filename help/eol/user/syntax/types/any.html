Any datatype

The Any datatype is the root datatype of EOL and represents anything. Therefore, its methods apply to all primitive types, collections and model elements.

Built-in Operations (on the type)

isType(element : Any) : Boolean Checks if the argument is of the type
isKind(element : Any) : Boolean Checks if the argument is of a compatible type
createInstance() : Any Creates a new instance of the type

Examples

Integer.isType(1) examines if 1 is of type Integer and returns true
String.createInstance() creates and returns a new empty String

Built-in Operations (on instances of the type)

print() : Any Prints the string representation of the instance in the default output stream and returns the instance
println() : Any Prints the string representation of the instance, followed by a new line, in the default output stream and returns the instance
isTypeOf(type) : Boolean Checks if the instance is of the specified type
isKindOf(type) : Boolean Checks if the instance is of a type compatible with the specified type
asSequence() : Sequence Returns a new Sequence containing the instance only. If the instance is of Collection type, the new Sequence contains all the items of the collection
asSet() : Set Returns a new Set containing the instance only. If the instance is of Collection type, the new Sequence contains all the unique items of the collection
asCollection() : Collection Returns a new Collection containing the instance only. If the instance is of Collection type, the new Collection contains all the items of the collection
asOrderedSet() : OrderedSet Returns a new OrderedSet containing the instance only. If the instance is of Collection type, the new OrderedSet contains all the unique items of the collection
isDefined() : Boolean Checks if the instance is defined (not null)
hasFeature(feature : String) : Boolean Checks if the instance supports a feature with the specified name

Examples

1.isTypeOf(Integer) examines if 1 is of type Integer and returns true
1.asSequence() creates and returns a new Sequence that contains only the Integer 1
